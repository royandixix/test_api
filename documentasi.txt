Options -Indexes: Menonaktifkan listing direktori.
<IfModule mod_rewrite.c>: Memastikan bahwa modul mod_rewrite tersedia dan aktif.
Options +FollowSymlinks: Mengizinkan server untuk mengikuti symbolic links.
RewriteEngine on: Mengaktifkan mesin penulisan ulang (rewrite engine).
RewriteCond %{REQUEST_FILENAME} !-f: Memeriksa apakah file yang diminta tidak ada (bukan file).
RewriteCond %{REQUEST_FILENAME} !-d: Memeriksa apakah direktori yang diminta tidak ada (bukan direktori).
RewriteRule ^([\s\S]*)$ index.php?$1 [L]: Mengarahkan semua permintaan yang tidak cocok dengan file atau direktori yang ada ke index.php.




















<IfModule mod_rewrite.c>: Memastikan bahwa modul mod_rewrite tersedia dan aktif.

RewriteEngine on: Mengaktifkan mesin penulisan ulang (rewrite engine).

RewriteRule ^$ public/ [L]:

^$: Mencocokkan URL root (misalnya, http://domain.com/).
public/: Mengarahkan URL root ke direktori public/.
[L]: Memberitahu server untuk berhenti memproses aturan lebih lanjut jika aturan ini cocok (flag "Last").
RewriteRule (.*) public/$1 [L]:

(.*): Mencocokkan semua permintaan URL.
public/$1: Mengarahkan semua permintaan ke subdirektori public/, di mana $1 adalah placeholder yang mewakili URL yang cocok.
[L]: Sama seperti di atas, ini adalah flag "Last" yang memberitahu server untuk berhenti memproses aturan lebih lanjut jika aturan ini cocok.
Kegunaan:
Kode ini berguna jika Anda ingin semua permintaan masuk diarahkan ke folder public/, sering digunakan dalam proyek yang menggunakan struktur direktori seperti Laravel, di mana semua file yang dapat diakses oleh publik ditempatkan di dalam folder public.
Kesimpulan:
Kode ini benar dan akan bekerja dengan baik untuk mengarahkan semua lalu lintas ke direktori public. Pastikan Anda memiliki folder public di dalam root proyek Anda, dan semua file publik atau index utama (index.php) berada di dalamnya.
